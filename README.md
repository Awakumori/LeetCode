27 移除元素
    双指针，lr一开始都指向起点；不断地试图用r的值覆盖l；如果r的值无需移除，那么就用r的值覆盖l；覆盖成功后，l向后移动；而r每一轮都向后移动。
    起初，lr同时指向首元素，假设前几个元素都无需移除，那么每一轮都会用自身覆盖自身，然后向后移动；直至遇到一个需要移除的元素，l就不动了，r向后移动，寻找一个无需移除的值，然后覆盖l。
